I have been using GNU Emacs for the last 4 years or so, since I
migrated from Neovim, after atleast a few attempts at figuring out
this juggernaut. I've moved most of my workflows into Emacs —
specifically using Org mode — and so it has become an almost integral
part of how I do nearly everything.

This is a collection of all the Emacs package(s) I've written and hope
to write over the coming years when I finally figure Emacs Lisp
out. This also includes my (il)literate Emacs configuration, which to
be honest is just a motley assembly of various snippets I've stolen
from others. I'm still in the process of learning elisp so please bear
with my (potentially) horribly written code! You can also view these
on my Sourcehut [[https://sr.ht/~peregrinator/][projects page]].
